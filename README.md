# nat-media-video

## Installation
```
weexpack plugin add nat-media-video
```

```
npm install weex-nat --save
```

## Usage

Use in weex file (.we)

```html
<script>
import 'Nat' from 'weex-nat'

// play
Nat.video.play('http://domain.com/video')

// pause
Nat.video.pause()

// stop
Nat.video.stop()

</script>
```

See the Nat [Documentation](http://natjs.com/) for more details.
